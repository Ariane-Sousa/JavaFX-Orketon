# Orketon

Aplicação de um Portal de Vagas. Este portal poderá ser acessado pelo candidato, que terá as seguintes opções: visualizar vagas em aberto, aplicar nas vagas e visualizar o andamento das vagas aplicadas e poderá ser acessado pelo funcionário, que poderá avaliar os candidatos aplicados nas vagas e cadastrar, editar e excluir vagas.
